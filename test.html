Decode Ways.Link:https://oj.leetcode.com/problems/decode-ways/ A message containing letters from A-Z is being encoded to numbers using the following mapping: 'A' -> 1 'B' -> 2 ... 'Z' -> 26 Given an encoded message containing digits, determine the total number of ways to decode it. For example, Given encoded message "12", it could be decoded as "AB" (1 2) or "L" (12). The number of ways decoding "12" is 2. class Solution: # @param s, a string # @return an integer def numDecodings(self, s): if s is None or len(s) == 0: return 0 if len(s) == 1: return 1 if 1 <= int(s) <= 9 else 0 k = len(s) - 2 x = 1 y = 1 if 1 <= int(s[k + 1]) <= 9 else 0 while k != -1: tmp = 0 if 1 <= int(s[k]) <= 9: tmp += y if 10 <= int(s[k:k + 2]) <= 26: tmp += x x = y y = tmp k -= 1 return y
